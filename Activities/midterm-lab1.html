Midterm Lab Task 1

Basic Python Exercises (Problems 1–4)

Description

This activity contains four basic Python problems. Each problem focuses on different concepts such as input/output, formatted printing, and simple logic using conditional statements.

Problem #1

Displays basic user information using print statements.

Problem #2

Shows how to use variables, strings, and formatted output for structured text display.

Problem #3

Demonstrates input collection and string formatting for displaying book information.

Problem #4

Shows conditional statements to display the day of the week based on user input.

Reflection

In this activity, I learned how to organize simple Python scripts and display structured output. It helped me become more familiar with variables, formatted printing, and decision-making using conditional statements.
